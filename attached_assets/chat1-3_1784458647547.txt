PROJECT SUMMARY
---------------
Project: Multi-tenant HRM + Distributor ERP (Blueprint is the single source of truth)

Goal:
Build a production-grade SaaS platform that is completely blueprint-compliant and deployment-independent (NOT Replit-dependent). Replit is only the development environment.

Core Principles:
- Follow the blueprint exactly whenever possible.
- If Replit constraints differ (Docker/Prisma/etc.), adapt implementation without changing architecture.
- Never hardcode secrets or cloud credentials.
- Every external service (AWS S3, Email, Redis, etc.) must exist as configurable integrations in backend settings.
- Multi-company architecture from day one.
- India (INR) as default locale, but system must remain globally deployable.
